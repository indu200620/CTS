# SQL Server Fundamentals

## Introduction

SQL Server is a Relational Database Management System (RDBMS) developed by Microsoft.

It is used to store, manage, retrieve, and secure data.

## SQL Server Editions

- Express Edition
- Developer Edition
- Standard Edition
- Enterprise Edition

## Key Components

- Database Engine
- SQL Server Agent
- Integration Services (SSIS)
- Reporting Services (SSRS)
- Analysis Services (SSAS)

## SQL Server Instance

An instance is a separate installation of SQL Server.

Types:

- Default Instance
- Named Instance

## SQL Server Management Studio (SSMS)

SSMS is a graphical tool used to:

- Create Databases
- Write Queries
- Manage Security
- Monitor Performance
- Backup and Restore Databases

# Views and Indexes

## Views

A View is a virtual table created using a SQL query.

### Benefits

- Security
- Data Abstraction
- Simplified Queries

### Operations

- Create View
- Modify View
- Update View
- Drop View

## Indexed Views

Stores data physically for faster retrieval.

## Indexes

Indexes improve query performance.

### Types

- Clustered Index
- Non-Clustered Index
- Unique Index
- Composite Index
- Covering Index

## Index Fragmentation

Occurs when data pages become inefficiently organized.

## Query Optimization

Improves execution speed by reducing resource usage.

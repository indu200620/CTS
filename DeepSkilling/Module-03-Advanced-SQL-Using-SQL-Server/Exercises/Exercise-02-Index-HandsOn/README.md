# Exercise 2 - Index Hands-On

## Objective

Create and use indexes to improve query performance.

## Problem Statement

The Employees table contains a large number of records. Create an index on the Salary column and execute a query using that column.

## Concepts Used

* Clustered Index
* Non-Clustered Index
* Query Optimization

## Learning Outcome

Learned how indexes improve search performance and reduce table scanning.

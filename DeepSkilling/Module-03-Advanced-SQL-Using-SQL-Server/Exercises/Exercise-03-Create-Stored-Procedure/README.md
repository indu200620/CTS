# Exercise 3 - Create Stored Procedure

## Objective

Create a stored procedure to insert employee records into the Employees table.

## Problem Statement

A company wants to automate employee insertion operations. Instead of repeatedly writing INSERT statements, a stored procedure should be used.

## Concepts Used

* Stored Procedures
* Parameters
* INSERT Statement

## Learning Outcome

Learned how to create reusable database procedures and pass parameters to perform operations.

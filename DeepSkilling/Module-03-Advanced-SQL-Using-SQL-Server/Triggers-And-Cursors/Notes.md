# Triggers and Cursors

## Triggers

A trigger automatically executes when an INSERT, UPDATE, DELETE or LOGON event occurs.

### Types

- AFTER Trigger
- INSTEAD OF Trigger
- LOGON Trigger

### Advantages

- Data Integrity
- Automatic Auditing

### Disadvantages

- Performance Overhead
- Difficult Debugging

## Cursors

A cursor processes records one row at a time.

### Uses

- Row by Row Processing
- Complex Business Logic

### Alternatives

- Joins
- CTE
- Window Functions

# Stored Procedures and User Defined Functions

## Stored Procedure

A stored procedure is a precompiled collection of SQL statements stored inside the database.

### Benefits

- Faster execution
- Code reusability
- Better security
- Reduced network traffic

### Types

- User Defined Stored Procedure
- Temporary Stored Procedure
- System Stored Procedure
- Extended Stored Procedure

## User Defined Functions

Functions return values and can be used inside queries.

### Types

- Scalar Function
- Table Valued Function
- Built In Functions

## Learning Outcome

Learned how to create reusable database logic using procedures and functions.

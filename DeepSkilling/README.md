# Module 3 - Advanced SQL Using SQL Server

## Overview

This module covers advanced SQL Server concepts used in enterprise database development and optimization.

## Topics Covered

### Advanced Concepts

- SQL Server Fundamentals
- OVER()
- PARTITION BY
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- GROUPING SETS
- ROLLUP
- CUBE
- Common Table Expressions (CTE)
- Recursive CTE
- MERGE
- PIVOT
- UNPIVOT

### Views and Indexes

- Creating Views
- Modifying Views
- Dropping Views
- Updating Views
- Indexed Views
- Clustered Index
- Non-Clustered Index
- Covering Index
- Query Optimization

## Learning Outcome

After completing this module, I can design optimized SQL queries, implement advanced SQL Server features, and improve database performance using views and indexes.

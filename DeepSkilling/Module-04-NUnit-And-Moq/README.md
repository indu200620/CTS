# Module 4 - NUnit and Moq

## Overview

This module introduces automated testing concepts using NUnit and Moq frameworks in .NET. Automated testing helps developers verify software functionality quickly and reliably. NUnit is a popular unit testing framework, while Moq is used to create mock objects for testing components that depend on external services.

## Learning Objectives

* Understand automated testing and its benefits.
* Learn Test Driven Development (TDD).
* Write unit tests using NUnit.
* Test methods, exceptions, arrays, and collections.
* Understand dependency injection.
* Create mock objects using Moq.
* Test interactions between software components.

## Topics Covered

1. Getting Started
2. Fundamentals of Unit Testing
3. Core Unit Testing Techniques
4. Breaking External Dependencies using Moq

## Outcome

After completing this module, learners will be able to create reliable unit tests and improve software quality using NUnit and Moq.
